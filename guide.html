After you install this app, That you finished setup and will be redirect to admin page to configure your app. The your list block will show the right column.

Note: images will were upload to asset of your store,so your store must allow for upload files

App configure Tutorial

After installed REGION RESTRICTIONS App you will be redirect to admin page.

We have 2 layout:
  • REGION RESTRICTIONS : Add location you want block, show list location blocked, edit and delete location. You can add the IP address or country's code you want to block in the input. In case add list ip or country's code, you just add a comma between them.
  • SETTTING LAYOUT: Setting and change display interface ( choose image background (if you use layout with background) and your store's logo). You can settings content you want:
    If you want to block all user accces your website choose Block All
    In case you do not choose BLock All. We have 2 list Black and White List. You can choose list you want to block. "Link redirect block" is link when user blocked which try to access.     Just below the settings content is Style Settings. To display Style Settings you just click settings content. You can change color or background for display interface.

How can I change background image or logo image?

  1. Go to setting layout
  2. Choose select your layout "Background"
  3. Browse to image to upload background or logo

Do I have to set error message?

You don't have to set that message. When visitors confirm wrong,the box dialog will shake to notice
